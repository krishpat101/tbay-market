## 📌 Project Overview
TBay Market is a full-stack classified ads web application for Lakehead University students to buy and sell items like textbooks, electronics, furniture, rides, and jobs — all in a secure, responsive platform.

---

## 🚀 Features
- User Authentication (secure login/register)
- Post Ads with images
- Search and filter listings
- Manage your ads
- Responsive UI (Bootstrap 5)

---

## 🔐 Security Features
- Password hashing (PBKDF2)
- SQL Injection protection
- Secure sessions
- File validation

---

## 🛠️ Tech Stack
- Backend: Python + Flask  
- Database: SQLite + SQLAlchemy  
- Authentication: Flask-Login + Werkzeug  
- Frontend: HTML + Bootstrap 5  
- Storage: Local filesystem (`static/uploads/`)  

## How to Run
pip install flask flask-sqlalchemy flask-login bcrypt pillow  
python app.py  
Open http://127.0.0.1:5000
