# TBay Market  
Thunder Bay's Student Classified Marketplace  

COMP 4431 / 4432 / 4434 — Winter 2026  
Krish Patel · Lakehead University  

## Project Overview
TBay Market is a full-stack classified ads web application for Lakehead students to buy and sell items like textbooks, electronics, furniture, rides, and jobs.

## Features
- User Authentication (secure login/register)
- Post Ads with images
- Search and filter listings
- Manage your ads
- Responsive design (Bootstrap 5)

## Security
- Password hashing (PBKDF2)
- SQL Injection protection
- Secure sessions
- File validation

## Tech Stack
- Python (Flask)
- SQLite
- SQLAlchemy
- Bootstrap 5

## How to Run
pip install flask flask-sqlalchemy flask-login bcrypt pillow  
python app.py  
Open http://127.0.0.1:5000
